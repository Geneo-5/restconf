# RESTCONF Qualification Test Plan

Ce dépôt contient le référentiel de qualification d'un serveur RESTCONF conforme aux RFC de l'IETF.

## Documents

| Document | Description |
|----------|-------------|
| 00_Introduction | Présentation générale |
| 01_Strategie_de_Test | Organisation des campagnes |
| 02_Infrastructure_RESTCONF | Validation protocolaire |
| 03_Ressources_RESTCONF | CRUD |
| 04_Parametres_RESTCONF | Query Parameters |
| 05_RPC_Actions_Notifications | RPC et Streams |
| 06_NMDA | Datastores |
| 07_Gestion_des_Erreurs | Erreurs et contraintes |
| 08_Securite | Validation sécurité |
| 09_Performance_et_Robustesse | Charge et stabilité |
| 10_Scenarios_End_to_End | Qualification fonctionnelle |
| 11_Matrice_de_Conformite | Couverture RFC |
| 12_Jeux_de_Donnees | Jeux de configuration |
| 13_Glossaire | Définitions |

Environ 350 cas de test couvrent les RFC8040, RFC8527, RFC7950, RFC7951, RFC6243 et RFC6415.
